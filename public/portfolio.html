Web3 GIF Portal built on Solana

This app is built on the Solana Blockchain and requires the user to connect their Phantom wallet. Once connected, the user then has access to the 'Fast and Furious' GIF Wall where they can view and submit their own favorite F&F GIF. After the user submits their GIF link the Phantom wallet prompts the user for their Transaction Signature and then sends the data to the blockchain, thus posting their GIF to the wall.

React and GraphQL Travel Blog

This travel blog was built to be mobile-first, even making the embedded YouTube a responsive-iframe. With a React.JS front-end framework and GraphQL content-management system, this blog loads quickly and can be stylishly customized with Tailwind CSS. The app utilizes GraphCMS and Vercell for the deployment.

Weather Dashboard

Simple dashboard app where the user can input a city name and see the current weather condition for that location, as well as a brief five day forecast. Once a city is searched it is saved into a list where it can be clicked to quickly change the displayed location to the city clicked. The weather information is sourced using Open Weather Map's API and integrated into the website utilizing jQuery.

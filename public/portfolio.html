Find-a-Fluff

MERN stack application utilizing two seperate APIs to provide the user with dog breed images and available adoptions, DogAPI, and Purina's Pet Adoption API. The front-end is built on React while the back-end utilizes MongoDB for user storage.

Kanye Quotes with Yoda

App that utilizes RESTful routes and Dual API integration to manipulate one response into a second query. Bootstrap is implemented for the styling on the front end while JavaScript handles the majority of on-screen effects. User is also able to manipulate the input text that is translated. This app was built with the help of a team using Agile methodology.

Google Book Search

MERN Stack app that allows a user to search for books using Google's search API, and sign-up/login to save books in a personalized collection. This app demonstrates refactoring skills by altering REST APIs into GraphQL with Apollo.

Weather Dashboard

Simple dashboard app where the user can input a city name and see the current weather condition for that location, as well as a brief five day forecast. Once a city is searched it is saved into a list where it can be clicked to quickly change the displayed location to the city clicked. The weather information is sourced using Open Weather Map's API and integrated into the website utilizing jQuery.

Tech Blog

This app provides the user with a CMS-style blog site, similar to Wordpress, where users can publish their blog posts and comment on ther posts from other users as well. This app follows the MVC paradigm in structure and utilizes Handlebars.js as the templating language, Sequelize as the ORM, and express-session for authentication. Users are required to create an account to have the ability to interact on posts.
